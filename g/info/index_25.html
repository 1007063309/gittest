添加或修改模块提示Unknown column mobile

出现问题的原因是部分手动更新的客户未按更新说明执行SQL语句，导致module表缺少mobile字段。

详见：https://www.destoon.com/product/release/173.html

按照更新说明在后台或者phpmyadmin等数据库管理软件里执行以下语句解决：

ALTER TABLE `destoon_module` ADD `mobile` VARCHAR( 255 ) NOT NULL AFTER `domain` ;
